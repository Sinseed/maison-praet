# Maison Praet

Site vitrine + CRM courtier immobilier.

## Démarrage

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Déploiement

Hébergé sur Vercel. Push sur `main` = déploiement automatique.

## Structure

- `/` — Site vitrine public
- `/crm` — Dashboard courtier (accès PIN)
